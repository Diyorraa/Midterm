# Midterm

## Get your ChatGPT api key or create a new one

Go to https://platform.openai.com/account/api-keys

## Set your ChatGPT api key

```bash
export OPENAI_API_KEY="YOUR_API_KEY"
```

## Install the package

```bash
pip install openai
```

## Run the program

```bash
python3 chatgptAPI.py
```
